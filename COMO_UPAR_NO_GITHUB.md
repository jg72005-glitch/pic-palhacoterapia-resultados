# Como subir no GitHub Pages

1. Entre em https://github.com e faça login.
2. Clique em `New repository`.
3. Nome sugerido: `pic-palhacoterapia-resultados`.
4. Crie o repositório como `Public` se quiser usar GitHub Pages gratuito.
5. Clique em `uploading an existing file`.
6. Arraste para lá todos os arquivos desta pasta: `index.html`, `.nojekyll`, `README.md` e `COMO_UPAR_NO_GITHUB.md`.
7. Clique em `Commit changes`.
8. Vá em `Settings` > `Pages`.
9. Em `Build and deployment`, selecione `Deploy from a branch`.
10. Em `Branch`, escolha `main` e `/root`, depois clique em `Save`.
11. Aguarde alguns minutos. O GitHub mostrará o link do site publicado.

Observação: esta versão é agregada e não inclui a base bruta dos participantes. A versão editável deve continuar apenas no computador.
